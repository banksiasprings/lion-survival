# Dossier — gameplay tweaks batch (2026-06-28)

**Trigger:** Steven, voice notes — a six-item wishlist. All edits in `index.html`.

## 1. Spear — toolbar slot 6
A flint-tipped spear: thrown like a rock but it hits **~3× as hard**. No stored ammo — it's crafted on
the throw from **2 wood + 1 rock**.

- HTML slot 6 (🗡️, cost hint `2🪵+1🪨`); `Digit6` selects it; `setSlot` highlight loop extended to 6;
  `onLeftClick` routes slot 6 → `throwSpear()`.
- `throwSpear()` gates on `woodCount ≥ 2 && rockCount ≥ 1`, decrements both, and pushes a projectile
  into the existing `thrownRocks` array tagged `{mult:3, spear:true}`. `buildSpearMesh()` is a shaft +
  cone tip laid along local −Z; the flight loop `lookAt`s the velocity so the tip leads the arc.
- `updateThrownRocks` now reads `m = r.mult||1`: lions `−m`, prey `−9·m`, gorilla `−8·m`. Rock throws
  push `mult:1`, so rock behaviour is unchanged. Kill-feed icon switches 🪨/🗡️.

## 2. Rock cap 5 → 10
`doCollect` pickup gate `rockCount<5` → `rockCount<10`.

## 3. Perching costs no stamina
Steven: staying in a tree shouldn't drain stamina.
- `PLAYER.perchDrain 7 → 0`; removed the perch-block drain **and** the `stamina ≤ 0 → dropFromTree()`
  forced-drop (no more "grip gave out").
- Stamina now **regens while perched** (dropped the `!player.inTree` guard on regen) — a tree is a safe
  place to catch your breath and wait out the pride or the gorilla.

## 4 & 5. Grass hides you — from the gorilla *and* the prey
The gorilla and the prey ignored the player's stealth signature; only the lions read it. Now they all do.
- **Gorilla:** `noticesPlayer = distP < GOR.PURSUE_RANGE * (treed ? 1 : stealth.visMul)`. Grass + crouch
  shrink its notice range (visMul ≈ 0.34 in grass+crouch → ~9 m from 26 m), and ducking into grass mid-
  chase drops you. A **treed** player stays the gorilla's exception — it always sees that one.
- **Prey:** the herd only treats the player as a threat when `pd < P.fleeDist * stealth.visMul`, so you
  can creep up through grass to set up the `[Q]` ×3 ambush instead of scattering them on approach.

Both reuse the existing `stealth.visMul` (computed in `updateStealth` before the AI updates each frame).

## 6. Calmer prey roam more
A grazing herd barely moved (`speed = s.speed * 0.16`, tiny random drift). Now the graze branch:
- walks at `s.speed * 0.42`, picks a fresh heading every 3–6 s,
- pauses head-down ~30 % of the time (`P.grazePause`, speed 0),
- looser herd cohesion (pull in only past 9 m, weight 0.5).

Net: an unthreatened herd drifts across the map. Measured ~11 u over 2 s vs ~2 u before.

## 7. Removed the "cigarette"

count 28 → 27; the refreshed `gorilla_render_upgrade.png` shows the clean muzzle.
